# DeviceTool
`UIDevice`扩展，方便获取设备的一些属性

# 特点
- 屏幕宽、高获取
- 是否是刘海屏手机获取
- 设备信息获取（比如获取手机机器名:iPhone 6s）

# 用法
- 刘海屏手机尺寸集合，兼容横屏

`GL_Notch_iPhone_Sizes`

- 刘海屏手机类型集合

`GL_Notch_iPhone_Types`

- 屏幕宽

`gl_width`

- 屏幕高

`gl_height`

- 设备机器名（比如获取iPhone 6s）

`gl_machineName`

- 设备信息

`gl_info`

- 是否是模拟器

`gl_isSimulator`

- 是否是刘海屏手机

`gl_isNotchiPhone`

- 刘海高度

`gl_notchHeight`

- 状态栏高度

`gl_statusBarHeight`

- 虚拟home键高度

`gl_homeIndicatorHeight`
